# Create Feature

Objetivo:

Implementar uma nova feature seguindo todas as regras do projeto.

Antes de iniciar:

Leia obrigatoriamente:

* .ai/project.md
* .ai/product/*
* .ai/architecture/*
* .ai/skills/*

Tarefa:

Implementar a feature solicitada.

Requisitos:

* Clean Architecture
* SOLID
* Repository Pattern
* Service Layer
* SQLAlchemy 2.0
* FastAPI

Gerar:

1. Model
2. Schema
3. Repository
4. Service
5. Router
6. Tests
7. Migration

Não gerar código fora da arquitetura definida.

Ao finalizar:

Explique:

* Arquivos criados
* Responsabilidades
* Fluxo da feature
