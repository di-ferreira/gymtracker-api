# GymTracker

## Visão do Produto

GymTracker é uma plataforma voltada para praticantes de musculação que desejam consultar exercícios físicos de forma rápida e organizada.

O sistema deve funcionar como um catálogo completo de exercícios.

Cada exercício deve possuir:

* Nome
* Descrição
* GIF
* Vídeo
* Instruções
* Equipamentos necessários
* Grupo muscular
* Grupo de movimento
* Exercícios substitutos

## Objetivos

Permitir que usuários:

* Naveguem por exercícios
* Busquem exercícios
* Filtrarem exercícios
* Consultem substituições
* Acessem conteúdo offline através do aplicativo

## Diferenciais

* Offline First
* Catálogo altamente visual
* Estrutura simples
* Baixo consumo de dados
* Alta velocidade

## Público-Alvo

* Iniciantes
* Intermediários
* Avançados

## Filosofia

Consulta rápida durante treinos.

Poucos cliques.

Pouca distração.

Informação objetiva.

## Papel da API

A API é responsável exclusivamente por:

* Disponibilizar catálogo
* Organizar conteúdo
* Versionar catálogo
* Permitir sincronização

A API não é responsável por:

* Treinos
* Usuários
* Autenticação
* IA em tempo de execução
